Tenants

Customer organizations exposed by the Central API for your credentials.

Introduction

Each row represents a tenant your API client can access. Data is read-only in this portal; tenant lifecycle changes still happen in Sophos Central.

Billing and filters

Facet filters (including billing type where available) help you focus on a subset. Dashboard shortcuts may land here with a filter already applied.
