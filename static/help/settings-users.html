Settings — Users

Local operator accounts for this installation.

Introduction

Administrators manage users who may sign in to the portal. Password hashes and profile fields live in the encrypted secrets database on the server—not in Sophos Central.

Roles

Admin users reach credential storage, sync controls, and destructive batch operations. User accounts can browse synchronized data but cannot change Central credentials or certain administrative settings.

Row actions

Use Add user to open the creation dialog. Existing rows may expose edit profile, password reset, role changes, or delete depending on your permissions. Deleting the account you are currently using forces a logout.
