Firewall groups

Central-managed groups and membership as reflected in your local sync.

Introduction

Firewall groups let you treat multiple appliances as a unit in Sophos Central. This view lists groups per tenant and shows sync-friendly metadata such as member counts and last refresh times.

Table and filters

Use Filters to limit rows by tenant or other facets. Search narrows the grid. Column visibility is controlled from the gear icon (Table columns for groups). Export downloads the current result set.

Creating groups

Authorized workflows can open Create group to define a name, pick a tenant, optionally seed configuration from an existing firewall, and assign members. See Create firewall group for field-level guidance.
