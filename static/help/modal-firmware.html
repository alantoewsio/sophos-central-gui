Firmware upgrades

Schedule or review Sophos Firewall firmware updates via Central.

Introduction

Upgrade dialogs call Sophos Central on your behalf. Available builds, eligibility, and scheduling rules come from Central—not from this app. Always confirm maintenance windows with your organization before applying production upgrades.

Single firewall

Open the upgrade action from the Firewall details panel or the firewalls grid. The modal lists target versions returned by the API. Pick a version, review prerequisites shown in the UI, and confirm. Some appliances may require approval or maintenance mode in Central.

Batch upgrade

When multiple rows are selected on the Firewalls tab, administrators can open the batch dialog. Each appliance can receive its own target version where the API allows. Optional scheduling fields let you defer execution to a planned time; clear the schedule to request immediate action where permitted.
