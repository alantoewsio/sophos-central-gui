Remove from local database

Administrator action on selected firewall rows.

Introduction

This confirmation dialog appears when you choose to delete selected firewalls from the local SQLite database used by SFOS Central Firewall Management.

What changes

After confirmation, the rows disappear from this application’s tables until the next sync. Custom fields stored only locally (such as map coordinates) are removed with the record.

Relationship to Sophos Central

The appliance is not deleted or unmanaged in Sophos Central. If Central still returns that firewall during sync, it will be recreated locally automatically.
