Export table data

Download the current grid as a file.

Introduction

Choose Export on supported toolbars to open a small popover. Pick either Excel (.xlsx) or CSV. The browser downloads the file immediately.

File formats

Excel workbooks open in Microsoft Excel or compatible tools and preserve column typing where possible. CSV is plain text—use it for scripting or systems that do not read XLSX.

What is included

Exports reflect visible columns, current sort order, active filters, and the rows currently loaded in the grid (not necessarily every record in the database). Load additional pages first if you need a larger slice.
