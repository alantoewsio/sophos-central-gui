Account — Change password

Update the password for the account you are using now.

Introduction

Choose Account → Change password in the user menu. Enter your current password once and the new password twice. Submitting updates the hash in the local secrets database.

Requirements

New passwords must satisfy the minimum length enforced by the server (shown on the form). Pick a unique secret; reuse of personal or corporate passwords is discouraged.
