Alert details panel

Full alert payload from the local database.

Introduction

Click an alert row on the Dashboard or inside the firewall inspector to open this flyout. It is a read-only view of the synchronized Sophos Central alert, including severity, timestamps, tenant, and descriptive text.

What you see

Fields mirror what the API stored during sync. If an attribute was empty in Central, it may be omitted here. Closing the panel returns focus to the originating table.
