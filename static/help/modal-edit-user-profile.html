Edit user profile

Administrators update another user’s directory fields.

Introduction

Open this dialog from the actions column in Settings → Users. It edits full name, email, and mobile number for the selected account without exposing their password.

Typical usage

Adjust fields as needed and choose Save. The table refreshes with new timestamps. Users can still override their own email and phone from Account → Profile unless your policy states otherwise.
