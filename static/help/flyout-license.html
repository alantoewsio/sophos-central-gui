License subscriptions panel

Detailed entitlement lines for a license summary row.

Introduction

From the Licenses tab, open this flyout to inspect individual subscription SKUs, quantities, and states as last synced from Sophos Central. The portal does not sell or assign licenses; it only displays API data.

Usage tips

Use the data to reconcile billing questions internally, then make changes directly in Sophos Central if updates are required. Large tenants may list many rows—scroll inside the panel or export the parent table for wider analysis.
