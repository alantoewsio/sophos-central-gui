Licenses

Subscription summaries synchronized from Sophos Central.

Introduction

The Licenses tab aggregates license usage and entitlement information per tenant. Values reflect the last successful sync for each stored credential.

Summary and details

Toggle between compact summary cards and a detailed table when the UI provides that control. Filters and search work the same way as on other tabs. Column layout is configured via Table columns for licenses.

Subscription drill-down

Where available, open a row or control to view subscription lines in the License subscriptions panel.
