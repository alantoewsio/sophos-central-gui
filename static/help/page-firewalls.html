Firewalls

Managed appliances synchronized from Sophos Central for the selected credentials.

Introduction

This table lists firewalls discovered through the Central API. Status, firmware availability, and badges (such as NEW or UPD) are derived from timestamps stored locally. The map above the grid shows the same population when coordinates exist.

Toolbar and filters

Filters opens facet controls for tenants, state, and other dimensions. Quick links under View apply common presets (for example online only or pending approval). Use the search field for free-text matching. The gear control opens Table columns so you can show or hide fields.

Export saves the visible columns and current filter context to a spreadsheet or CSV.

Row actions

Select a hostname or status cell to open the Firewall details panel with alerts, subscriptions, firmware options, and location editing.

Administrator actions

Administrators may see batch buttons when rows are selected: approve pending management, schedule firmware upgrades, or remove records from the local database only. Removing locally does not delete the appliance in Central; it can reappear on the next sync.
