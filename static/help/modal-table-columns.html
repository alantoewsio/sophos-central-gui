Table columns

Choose which fields appear in each main grid.

Introduction

Open the gear icon beside the search field on a tab to show this dialog. Toggle columns on or off and use the search box inside the dialog to find a field by name. Your choices are remembered for that tab in this browser.

Firewalls

Firewalls expose the widest set of API-backed attributes (status, firmware, interfaces, tags, and more). Hide columns you rarely need to keep the grid readable; required identifiers stay available through row actions and the detail panel.

Groups

Group columns focus on membership counts, sync metadata, and tenant association. Trim the list when comparing many tenants side by side.

Tenants

Tenant columns include identifiers and billing-related fields returned by Central. Enable only the attributes your workflow needs.

Licenses

License summary columns can be dense; use toggles to emphasize subscription names, quantities, or states relevant to your review.
