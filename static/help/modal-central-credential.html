Add Central credential

Register API client credentials for synchronization.

Introduction

Administrators launch this form from Add credential on the Credentials settings panel. It captures the OAuth client values created inside Sophos Central’s partner or customer API settings.

Fields

  • Display name — label shown in tables and sync screens.
  • Client ID — public identifier issued by Central.
  • Client secret — confidential string; never share it outside the secrets store.

Test and save

Run Test connection to obtain a token and verify scopes before you commit. Successful tests populate Central identity metadata. Add credential persists the row and enables scheduler jobs.
