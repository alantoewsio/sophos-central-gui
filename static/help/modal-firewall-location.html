Firewall location

Set latitude and longitude used on dashboard and firewall maps.

Introduction

Locations are stored in the local database and drive map markers. Geocoding uses the public Nominatim service (OpenStreetMap data). Results depend on address quality and third-party availability.

Address search

Type a street address or place name, run the search, then choose a result to populate the map preview and coordinate fields. Refine the query if multiple regions share the same name.

Coordinates

You may edit decimal latitude and longitude directly when you already have precise values. Save applies the point to the selected firewall and updates markers on the next refresh.

Privacy note: Address queries leave this application to reach the geocoding provider. Do not submit sensitive location details you are not permitted to share externally.
