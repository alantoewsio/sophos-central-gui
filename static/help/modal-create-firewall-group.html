Create firewall group

Wizard for registering a new group in Sophos Central from this portal.

Introduction

This modal collects everything Central needs to create a firewall group: owning tenant, display name, optional configuration seed, and member appliances.

Tenant and name

Select the tenant that will own the group, then enter a unique name. The form stays locked until a tenant is chosen so the API receives a valid scope.

Initial configuration

You can start from a blank template or import baseline settings from an existing managed firewall. When import is selected, pick a source appliance from the list—only approved, eligible firewalls appear.

Members

Use the dual lists to move firewalls into the group. Search narrows each side. Transfer buttons add or remove members in bulk. After submission, the new group appears following the next sync (or immediately when the API confirms creation).
