Settings — About

Build metadata for this deployment.

Introduction

The About panel is read-only. It helps operators confirm which package is running and which license terms apply to the software bundle they installed.

Displayed fields

Typical rows include the marketing application name, optional Python project name, short description, semantic version, and license summary sourced from pyproject.toml when present.
