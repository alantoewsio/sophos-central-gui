Settings — General

Application-wide UI timing and security options.

Introduction

The General section controls how long firewall rows qualify for highlight badges and how aggressively idle browser sessions are ended.

NEW and UPD badges

Show NEW (hours) defines how recent both creation and last state change must be for the NEW indicator. Show UPD (hours) covers firewalls that recently changed state but are not classified as NEW or OLD. Values are stored in the application database and apply after you click Save.

Session idle timeout

Set Session idle timeout (minutes) to sign users out after a period without HTTP activity. Use 0 to rely only on cookie expiration. Keyboard, mouse, and scroll activity in the app counts as usage.
