Dashboard

Overview of synchronized Sophos Central data for your stored API credentials.

Introduction

The Dashboard is the home view after you sign in. It combines a geographic map, high-level counters, and a searchable list of recent alerts pulled from your local database (updated on the sync schedule you configure in Settings).

Firewall map

The collapsible Firewall map plots firewalls that have coordinates. Markers reflect the data from your last successful sync. If a firewall has no location, open it from the Firewalls list and set coordinates in the location dialog.

Summary cards

Cards summarize counts such as online and offline appliances, license-related totals, and other aggregates. Many cards link to a filtered view on another tab so you can drill in with one click.

Alerts

Use Filters to narrow alerts by tenant, firewall, severity, and other facets. Search matches visible columns. Export downloads the current result set. Rows per load controls how many rows are requested at a time as you scroll or load more.

Open a row to inspect full alert text and metadata in the Alert details panel.
